Beautiful shell prompt themes that perfectly complement your Neovim colorscheme.

## Available Shells

- **bash/** - Bash prompt with Git integration and color exports
- **fish/** - Fish shell with syntax highlighting and prompt
- **zsh/** - ZSH theme with advanced prompt and Git status

## Installation

### Bash

Add to your `~/.bashrc`:

```bash
source ~/path/to/Lazord/shells/bash/lazord.bash
```

### Fish

Add to your Fish configuration (`~/.config/fish/config.fish`):

```bash
source ~/path/to/Lazord/shells/fish/lazord.fish
```

### ZSH

Add to your `~/.zshrc`:

```bash
source ~/path/to/Lazord/shells/zsh/lazord.zsh
```

## Features

- Git branch and status indicators
- Directory path highlighting
- Error code visualization
- Consistent color scheme with Neovim
- Performance optimized prompts

- **Blue**: `#2176c7`
- **Green**: `#738a05`
- **Yellow**: `#a57706`
